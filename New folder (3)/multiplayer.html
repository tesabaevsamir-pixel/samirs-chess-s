🎮 How to Play with Friends:

  1. Choose "Online Multiplayer" mode
  2. Player 1 clicks "Create Room" and shares the room code
  3. Player 2 enters the room code and clicks "Join Room"
  4. Exchange connection info through the prompts
  5. Start playing! Take turns making moves
